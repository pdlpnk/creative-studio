# Data Model

## Creative Concept

Creative Concept — центральная сущность Creative Studio. Она хранит креативное решение и связывает бриф, производство, тестирование и обучение.

| Поле | Назначение |
|---|---|
| `concept_id` | Уникальный идентификатор концепции в формате `CC-0001`. |
| `name` | Краткое понятное название концепции. |
| `geo` | Целевая география или рынок. |
| `campaign_type` | Тип кампании или рекламного сценария. |
| `objective` | Измеримая цель концепции. |
| `offer` | Предложение, продукт и существенные условия. |
| `audience` | Описание целевой аудитории и её контекста. |
| `style` | Визуальный стиль или ссылка на запись в Brain. |
| `season` | Сезонный контекст или период актуальности. |
| `composition` | Принципы расположения и визуальной иерархии. |
| `copy` | Текстовые сообщения, заголовки и призывы к действию. |
| `visual_brief` | Требования к будущему визуалу и ограничения производства. |
| `hypothesis` | Проверяемое предположение, лежащее в основе концепции. |
| `status` | Текущий этап жизненного цикла концепции. |
| `parent_concept_id` | Идентификатор родительской концепции для вариации; иначе пустое значение. |
| `created_at` | Дата и время создания в формате ISO 8601. |
| `test_result_id` | Ссылка на связанный результат теста, если он существует. |
| `learning_tags` | Набор нормализованных тегов для поиска и повторного использования знаний. |

На текущем этапе документ задаёт логическую модель, а не конкретный формат хранения. Обязательность полей, допустимые статусы и справочники будут уточняться перед реализацией.
