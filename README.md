# Creative Studio

Creative Studio — внутренняя AI-система управления креативным производством. Она помогает превращать брифы и гипотезы в структурированные креативные решения, поддерживать производство визуалов и накапливать знания по результатам тестов.

Центральная сущность системы — **Creative Concept**: единое описание идеи, аудитории, оффера, визуального направления, текста, гипотезы и последующих результатов. Концепт связывает стратегию, производство, тестирование и обучение.

## Основные модули

- **Brain** — база знаний о географиях, стилях, сезонах, композициях, типографике и накопленных выводах.
- **Factory** — будущий производственный контур: роли, промпты, шаблоны, скрипты и процессы создания материалов.
- **Learning** — контур анализа тестов и возврата проверенных выводов в Brain.

## Текущий статус

Проект находится на стадии подготовки фундамента. Созданы структура репозитория и базовая документация; приложение, интеграции, генерация изображений и автоматизация пока не реализованы.

## Структура репозитория

```text

├── prompts/               # Будущая библиотека промптов

```

Правила хранения старых креативов описаны в [Creative Reference Library](assets/references/README.md).
